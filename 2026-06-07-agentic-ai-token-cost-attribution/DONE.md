## What was built

ATCAP is a fully functional AI Token Cost Attribution Platform with a Python SDK that wraps OpenAI/Anthropic clients to capture token usage with business context (team, feature, workflow, entity ID), a FastAPI backend that computes real-time costs against a versioned pricing catalog, evaluates budget policies with Slack alerting, and correlates spend against business value events. The React dashboard with Recharts visualises cost breakdowns by team/feature/model, budget utilisation, ROI correlation records, and alert management — and a demo script seeds 30 days of synthetic data to demonstrate the full end-to-end attribution flow out of the box.
